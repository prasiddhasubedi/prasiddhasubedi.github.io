# Literary Portfolio

A personal literary portfolio website showcasing poetry, articles, ebooks, and photography.

## Repository

This repository has been renamed from `byprasiddha.com` to `byprasiddha`.

**New Repository URL:** https://github.com/prasiddhasubedi/byprasiddha

## Setup

To run this website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/prasiddhasubedi/byprasiddha.git
   ```

2. Open `index.html` in your web browser

## Dependencies

This project uses the following dependencies:
- Framer Motion (for animations)
- GSAP (for premium animations)
- AOS (Animate On Scroll library)
- Google Fonts (for typography)

To install npm dependencies:
```bash
npm install framer-motion
```

## Features

- **Poetry Collection**: Curated collection of poems
- **Articles**: Thought-provoking pieces on culture, technology, and life
